エラーコード一覧

カスタムインフィード広告のロードと各種通知時のイベント取得を利用してイベントを受け取る際のエラー詳細についての説明です。

エラーコード 説明
1000 対象の広告枠にて広告の在庫切れなどによる広告取得が失敗したことを示す。
1001 Media IDが設定されていないことを示す。SDKの初期化をご確認ください。
1002 SDK内部にエラーが発生したことを示す。担当者までご連絡ください。
1003 設定されたMedia IDが無効のことを示す。
1004 SDKの初期化が未完了のことを示す。少々待ってから広告取得を再度呼んでください。
1005 広告を表示するUITableViewが設定されていないことを示す。
1006 不正なパラメータが設定されたことを示す。
1007 同じ広告リクエストが重複されたことを示す。
