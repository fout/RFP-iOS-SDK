HTTPリクエストのタイムアウト間隔の設定

(v2.6.0以上利用可能)

RFP.rfpSetHttpRequestTimeoutInterval(_:)を利用することで、SDKのHTTPタイムアウト値を設定することが可能です

func application(_ application: UIApplication, didFinishLaunchingWithOptions launchOptions: [UIApplicationLaunchOptionsKey: Any]?) -> Bool {

    // HTTP requestのタイムアウトを15秒にする
    RFP.rfpSetHttpRequestTimeoutInterval(15)

    // ※ rfpSetHttpRequestTimeoutIntervalは必ずrfpInitMediaの前に設定をしてください
    RFP.rfpInitMedia("your_media_id")

    // 現在の設定値を取得
    let timeoutInterval = RFP.rfpGetHttpRequestTimeoutInterval()
    // ...
}

タイムアウト間隔設定時の注意点

  • 極めて短い間隔を設定すると広告を取得できない可能性がありますので、使用状況に合わせて設定してください。

  • システム最大値を超えた値が設定された場合、システム最大値が適用されます。
