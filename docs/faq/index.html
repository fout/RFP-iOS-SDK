名称 API 詳細

OS バージョン UIDevice systemVersion 5.0.1, 7.1 など
端末名称 utsname machine iPhone4,1, iPhone7,1 など
言語設定 NSLocale preferredLanguages [0] ja, en など
国コード NSLocale NSLocaleCountryCode JP, US など
